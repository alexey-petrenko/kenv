Graphical User Interface
========================

The installation requires the `PyQt <https://wiki.python.org/moin/PyQt>`_.

**Installation steps**:

-  Install ``PyQt5``

   ::

       pip install pyqt5


-  Install ``kenv_gui``

    Install from the souces, by cloning the `code from Github <https://github.com/fuodorov/kenv/tree/master/gui>`_.
