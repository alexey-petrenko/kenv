The Equations class
======================

The :any:`Equations` class is top-level class in KENV. It contains all
the equations data, and has the high-level methods :any:`envelope_prime`, :any:`centroid_prime`, :any:`particle_prime`
that performs the integrate.

   :members: envelope_prime, centroid_prime, particle_prime
